GIOŚ API JavaScript Wrapper

gios-api is JavaScript wrapper of GIOŚ API

Install

Yarn

yarn add gios-api

Npm

npm install gios-api

Import and use

import { GIOS_API } from "gios-api";

const giosAPI = new GIOS_API();
const stations = await giosAPI.findAll();
